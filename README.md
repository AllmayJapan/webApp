# **CI/CD 練習用のためのwebアプリプロジェクト**
---
- ## このプロジェクトはGitHub Actionsを使用したCI/CDをマスターするためのプロジェクトです
    * webアプリ制作を通してGitによるバージョン管理とその他基本的な操作、
    GitHubでのリモートリポジトリの操作を復習します。

- ## プロジェクトボードも使用しつつ進めていこうかなと思います。
---
リポジトリのクローンは
```bash
gh repo clone AllmayJapan/webApp
```
issueの作成は
```bash
gh issue create
```
